4. Створення посту
BeforeMethod
- створити юзера

Steps
- відкрити сторінку логіну
- ввести username в поле Username (на формі логіну)
- ввести password в поле Password (на формі логіну)
- натиснути на кнопку Sign In

- ввести Title
- ввести Body Content
- натиснути на кнопку Save New Post

Assertions
- відображається New post successfully created.

BeforeMethod

- натиснути на кнопку Sign In

- відкрити створений Post
- натиснути на кнопку Edit
- ввести новий Title
- ввести новий Body Content
- проставити Is this post unique? == true
- обрати Сообщение для группы
- натиснути на кнопку Save Updates
- натиснути на кнопку Profile
- відкрити відредагований Post

Assertions
- відображається Post successfully updated. (одразу після редагування)
- відображаються нові Title та Body Content
- пост для Group Message
- Is this post unique? : yes
- відображаються кнопка Edit та Delete

AfterMethod
- видалити пост
- видалити юзера
